## About This Repository

This repository documents my work through the Cognizant Digital Nurture 5.0 program — a 7-week deep skilling track covering the full DotNet FSE stack. Every folder here corresponds to a week of the program, and contains the exercise solutions I wrote and tested myself.

The program covers a lot of ground fast — from design patterns and DSA in the first two weeks, through SQL, Entity Framework, ASP.NET Core Web API, Angular, and finally microservices and DevOps in week 7. The solutions are all in C# (.NET 8), built in Visual Studio Community Edition.

## What I've Covered So Far

**Week 1 — Design Patterns & SOLID**  
Worked through all five SOLID principles with C# examples, then implemented the main GoF patterns — Singleton, Factory, Builder on the creational side, Adapter, Decorator, Proxy for structural, and Observer, Strategy, Command for behavioral. Also practiced Dependency Injection and MVC.

**Week 2 — Data Structures & Algorithms**  
Implemented singly, doubly, and circular linked lists from scratch. Covered binary and linear search, and sorting algorithms (Bubble, Quick, Merge) with Big-O analysis for each.

**Week 3 — Advanced SQL (in progress)**  
Window functions using OVER(), PARTITION BY, ROW_NUMBER, RANK, and DENSE_RANK. Common Table Expressions including recursive CTEs. Stored procedures, user-defined functions, triggers, cursors. Transactions with ACID properties and isolation levels.

**Prerequisites:**
- Visual Studio Community 2022 or later

```bash
cd Week1-DesignPatterns/SOLID-Principles/

```
